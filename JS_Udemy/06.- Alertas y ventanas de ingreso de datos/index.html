Alertas y ventanas

Existen tres funciones simples para hacer alertas o ventanas emergentes:

  • Ventanas emergentes
  • Ventanas de confirmación
  • Ventana de ingreso de datos

¿Qué es una función?

Una función se puede invocar todas las veces que se desee y es utilizado para reutilizar el bloque de instrucciones que contengan.
A las funciones como las ventanas emergentes le podemos pasar datos como strings.
