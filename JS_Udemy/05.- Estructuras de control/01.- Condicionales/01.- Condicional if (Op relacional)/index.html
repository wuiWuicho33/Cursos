Condicional if (Operadores relativos)

El condicional if es la estructura de control más básica que no permite hacer una condición, if significa si, ya que de cumplirse o no una condición dentro de ella se ejecuta un bloque de instrucciones para hacer algo, no hacer nada o hacer otra cosa.

Un condicional es una estructura de control que nos permite comparar valores o hacer una condición concreta.

Ejemplo: si A es (igual, diferente, mayor, menor) a B entonces haz algo.
