Bucle while

While significa mientras, ya que mientras se cumpla la condición se ejecutará el bloque de código.
Es parecido al bucle for pero menos estricto, es un bucle indefinido y podemos hacer bucle infinitos.

Cómo funciona

Consta de una condición que al ser cumplida se ejecuta un bloque de instrucciones infinitamente, para parar el ciclo podemos colocar un contador o algo para que en algún momentola condición deje de cumplirse.

Sintáxis

while (condición) {
// Instrucciones a ejecutarse
}

while (year <= 2050) {
console.log("Estamos en el año " + year);
}
