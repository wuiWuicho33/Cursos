Bucle Do while

A diferencia de un bucle normal que primero se evalia una condición y de cumplirse se ejecuta un código, el bucle do while es lo contario, permite ejecutar un bloque de instrucciones de primera instancia y después se comprueba la condición, de cumplirse esa condición se ejecuta de nuevo o se detiene en la primera ejecución.

Sintáxis

do {
// Bloque de instrucciones
} while ( // Condición)
