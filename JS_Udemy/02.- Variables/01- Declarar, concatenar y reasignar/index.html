Variables

Una variable es un contenedor que sirve para almacenar información, es como si en una caja guardaramos un dato.

¿Cómo declaramos una variable?

A la palabra reservada var le asignamos un nombre y guardamos un dato en ella entre comillas.

Siempre hay que poner a la variable un nombre claro y descriptivo.

Concatenar cadenas de texto

El signo de más en javascript tambien se ocupa para concatenar cadenas y no sólo hacer operaciones, en otros lenguajes para concatenar se usa el punto (.), pero en javascript el punto sirve para acceder a la propiedad de un objeto.

Reasignar valores

No es necesario escribir la palabra reservada var nuevamente ya que el nombre de la variable ya ha sido declarado, sólo se escribe el nombre de la variable y se le asigna un nuevo valor.
