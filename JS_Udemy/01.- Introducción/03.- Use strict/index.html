Let

Al igual que la palabra reservada var, let tambien funciona para declarar variables, es una palabra proveniente del tipescript que ya es aceptado por el emmascript

Let permite definir variables limitando su alcance al bloque, declaración o expresión donde se esta usando.

Var define una variable global o local de acuerdo a una función sin importar el ámbito del bloque

La diferencia entre Let y Var es el alcance que tienen las variables y cómo actuán en función de los diferentes bloques

En resumen, las variables declaradas con let tienen un alcance limitado al bloque de código donde es declarado, mientras que var define variables globales que se reemplazaran si volvemos a declarar la misma variable pero con un nuevo valor.
